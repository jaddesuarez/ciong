# CIONG - Celebrating the Value of Friendship

CIONG is an NGO project dedicated to exploring and promoting the fundamental value of friendship in our society.

## 🚀 Project Overview

The project consists of several key sections that explore different dimensions of friendship:

- **Hero Section**: Welcoming introduction to the concept of friendship
- **What is Friendship**: Fundamental exploration of friendship's meaning
- **Essence of Friendship**: Core values and principles that define true friendship
- **Types of Friendship**: Different categories and forms of friendship
- **Qualities**: Essential characteristics that make strong friendships
- **Resources**: Additional materials and references about friendship

## 🛠️ Technologies Used

This project is built with modern web technologies:

- [Astro](https://astro.build/) - The main web framework
- [TailwindCSS](https://tailwindcss.com/) - For styling and responsive design

## 📋 Prerequisites

Before you begin, ensure you have the following installed:

- [Node.js](https://nodejs.org/) (.nvmrc version)

  Simply execute:

  ```
  nvm use
  ```

- npm (comes with Node.js)

## 🔧 Setup Instructions

1. Clone the repository:

```bash
git clone [https://github.com/jaddesuarez/ciong.git]
cd ciong
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```

5. Preview the production build:

```bash
npm run preview
```
